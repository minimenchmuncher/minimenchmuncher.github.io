> This blog is about data analysis and structuring and designing data analyses.
> I try to use open-source code when possible and keep what I'm writing
> available for everybody.

Hello and welcome to my blog.

This blog is my random musings on data analysis, links to my projects,
that sort of thing. The projects on this blog are all open-source, and I
aspire to use all industry best-practices, including a number of my own
found best practices over the years. There are links to all available
data, process, any code I use, with some sprinkles of why I use the tools
that I use. Thank you for reading, I welcome all comments and suggestions!

This blog is powered by [Jekyll](https://jekyllrb.com), and using the [Hyde Theme](https://github.com/poole/hyde).
